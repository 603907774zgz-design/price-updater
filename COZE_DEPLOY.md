## ⚙️ 标准表数据导入

在扣子平台中，标准表数据需要预先导入。

### 方式1: 使用扣子知识库
1. 创建知识库
2. 上传标准SKU表格
3. 在工作流中查询知识库

### 方式2: 使用扣子数据库
1. 在扣子平台创建数据库
2. 导入标准SKU数据
3. 在代码中连接数据库

### 方式3: 使用多维表格（推荐）
1. 保持现有的飞书多维表格
2. 在扣子工作流中添加「多维表格」节点
3. 通过 API 读取和写入数据

## 🔍 故障排查

### Bot 无响应
- 检查飞书应用权限是否添加完整
- 检查 Bot 是否已发布
- 查看扣子平台日志

### 识别失败
- 检查文本格式是否符合预期
- 查看扣子日志中的错误信息
- 确认标准表数据已正确导入

### 价格未更新
- 检查数据库连接是否正常
- 确认 SKU 匹配逻辑是否正确
- 查看价格历史记录
